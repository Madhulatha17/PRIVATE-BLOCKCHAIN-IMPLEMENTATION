# DESCRIPTION 

# Project Overview  

This project demonstrates the setup of a **private blockchain network using Ethereum (Geth)** and deployment of a simple decentralized application (DApp).

The system allows users to store and retrieve data securely on a private blockchain.



#  Objectives  

- Set up a private blockchain network  
- Deploy a smart contract on the private chain  
- Build a simple decentralized application (DApp)  
- Enable secure blockchain-based data storage and retrieval  


# Features  

- Private Ethereum blockchain setup  
- Custom genesis configuration  
- Smart contract deployment  
- Simple storage functionality  
- Web-based DApp interaction  



#  Smart Contract  

The smart contract allows:

- Storing data on blockchain  
- Retrieving stored data  
- Secure decentralized interaction  



#  Technologies Used  

- Ethereum (Geth)  
- Solidity  
- Web3.js  
- Remix IDE  
- MetaMask  



#  Deployment Steps  

- Initialize private blockchain using genesis file  
- Start Ethereum node using Geth  
- Deploy smart contract using Remix IDE  
- Connect MetaMask to local network  
- Run DApp in browser  
- Interact with smart contract
